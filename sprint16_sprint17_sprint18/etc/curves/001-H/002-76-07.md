# Хипербола — каноническо уравнение през две точки

## Дадено

Хиперболата минава през $A\!\left(5,\ \dfrac{4}{3}\right)$ и $B\!\left(\sqrt{34},\ -\dfrac{5}{3}\right)$.

## Случай 1: Фокуси върху $Ox$

Каноничното уравнение е $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$. Нека $u = \dfrac{1}{a^2}$, $v = \dfrac{1}{b^2}$.

Заместваме координатите на двете точки:

$$25u - \frac{16}{9}v = 1$$

$$34u - \frac{25}{9}v = 1$$

Умножаваме по 9:

$$225u - 16v = 9$$

$$306u - 25v = 9$$

От първото уравнение:

$$v = \frac{225u - 9}{16}$$

Заместваме във второто:

$$306u - 25 \cdot \frac{225u - 9}{16} = 9$$

$$4896u - 5625u + 225 = 144$$

$$-729u = -81 \implies u = \frac{1}{9} \implies a^2 = 9$$

$$v = \frac{225 \cdot \frac{1}{9} - 9}{16} = \frac{25 - 9}{16} = 1 \implies b^2 = 1$$

$$\boxed{\frac{x^2}{9} - y^2 = 1}$$

**Проверка:**
- $A\!\left(5,\ \frac{4}{3}\right)$: $\dfrac{25}{9} - \dfrac{16}{9} = \dfrac{9}{9} = 1$ ✓
- $B\!\left(\sqrt{34},\ -\frac{5}{3}\right)$: $\dfrac{34}{9} - \dfrac{25}{9} = \dfrac{9}{9} = 1$ ✓

## Случай 2: Фокуси върху $Oy$

Каноничното уравнение е $\dfrac{y^2}{a^2} - \dfrac{x^2}{b^2} = 1$. Нека $u = \dfrac{1}{a^2}$, $v = \dfrac{1}{b^2}$.

Заместваме координатите на двете точки:

$$\frac{16}{9}u - 25v = 1$$

$$\frac{25}{9}u - 34v = 1$$

Умножаваме по 9:

$$16u - 225v = 9$$

$$25u - 306v = 9$$

От първото уравнение:

$$u = \frac{225v + 9}{16}$$

Заместваме във второто:

$$25 \cdot \frac{225v + 9}{16} - 306v = 9$$

$$5625v + 225 - 4896v = 144$$

$$729v = -81 \implies v = -\frac{1}{9}$$

Тъй като $v = \dfrac{1}{b^2} < 0$, няма реално решение — **случаят е невъзможен**.

## Заключение

Единственото каноническо уравнение е:

$$\boxed{\frac{x^2}{9} - y^2 = 1}$$
