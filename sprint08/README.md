#### Геометрична прогресия

1. Дадена е геометрична прогресия $a_1, a_2, ..., a_n, ... $. 
    
    а) Ако първият член е $u$, а частното е $v$ , докажете, че  $a_1 + a_2 + ... + a_{n-1} = u\frac{1-v^n}{1-v}$

    - Упътване: използвайте формулата  $1-x^n = (1-x)(1 + x + x^2 + ... + x^{n-1})$

    б) Ако в допълнение $-1 < v < 1$, Докажете че $\lim_{n \to \infty}a_1+a_2+...+a_n = u\frac{1}{1-v}$

    - Упътване: използвайте доказаното от първата подточка и $\lim_{n \to \infty} v^n = 0$
