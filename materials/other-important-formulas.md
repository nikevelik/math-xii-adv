# Други важни формули, които е добре да се знаят

1. Твърдение 1. 

    Права s: Ax + By + C = 0 е допирателна към елипса epsilon: x^2/a^2 + y^2/b^2 = 1, ако A^2a^2 + B^2b^2 - C^2 = 0

2. Твърдение 2. 

    Права s: Ax + By + C = 0 е допирателна към хипербола xi: x^2/a^2 - y^2/b^2 = 1, ако A^2a^2 - B^2b^2 - C^2 = 0.
